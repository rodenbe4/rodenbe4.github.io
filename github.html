  • eScavenge
     This is the group project we completed as part of our software engineering course, implemented in three two-week sprints. It works as a scavenger hunt styled after the TV show The Amazing Race, where teams compete to reach specific destinations hinted at by clues, and need to be the first team to reach the final destination to win.
     Our program allows for two different end users: a "gamemaker", who is in charge of creating the game itself and position the landmarks, clues, and hints; and the "team" who is responsible for answering the questions about each landmark correctly before being allowed to move on to the next landmark. The team who reaches the last landmark first and correctly answers its question is the winner, although the rest of the teams may continue playing until they either finish or forfeit.

  • Semester Schedule
     In our System Programming class, we created a program using C++ that will allow a user to create a course schedule for a given semester. The assignment was designed to utilize C++ classes, as well as file I/O. Each class has a course code, days of the week met, a time period, and a teacher. The program allows the user to be able to add a class, remove a class, validate the current schedule to make sure that no teacher is teaching any two courses at the same time, and clear the schedule in its entirety. The user can also import a schedule from a file, or export a schedule into its own file.
